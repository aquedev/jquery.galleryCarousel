Gallery Carousel options and their defaults.

Requirements:

  • jQuery v1.5.2 or later: http://docs.jquery.com/Downloading_jQuery#Download_jQuery
  • jQuery cycle plugin: https://raw.github.com/malsup/cycle/master/jquery.cycle.all.js
  • jQuery jCarousel: http://sorgalla.com/jcarousel

Usage:

Initiate the plugin with the call below and use the markup in this demo. Any cycle plugin options can be passed using the options parameter: 'cycleSettings: {}'. Available jQuery cycle options. Available jQuery cycle fx options. 

$(function(){
	$('#targetElement').galleryCarousel({});
});

Options:

					
{
	activeItemClass: 'scroller-image-selected',
	height: 280,
	width:null,
	auto:false,
	cycleSettings: {}
}
